WebRTC终端实验项目

***感谢伟大的[PION](https://github.com/pion/webrtc)项目***

# 特性
+ 网关
+ 支持Windows和Linux
+ typescript
+ pure go（纯GO）

## 安装
	go get -u github.com/vlorc/webtty
